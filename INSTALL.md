# Install AstroMuninn

AstroMuninn is distributed as prebuilt CLI archives.

## macOS Apple Silicon

1. Download `AstroMuninn-vX.Y.Z-macos-apple-silicon.tar.gz` from the Releases page.
2. Extract the archive.
3. Move `astromuninn` somewhere on your `PATH`, for example `/usr/local/bin` or `~/bin`.
4. Run `astromuninn --help`.

## Windows x86_64

1. Download `AstroMuninn-vX.Y.Z-windows-x86_64.zip`.
2. Extract the zip archive.
3. Place `astromuninn.exe` in a directory you keep for CLI tools.
4. Add that directory to `PATH` if needed.
5. Run `astromuninn.exe --help` from PowerShell or Command Prompt.

## Linux x86_64

1. Download `AstroMuninn-vX.Y.Z-linux-x86_64.tar.gz`.
2. Extract the archive.
3. Move `astromuninn` to a directory on your `PATH`, such as `/usr/local/bin` or `~/bin`.
4. Run `astromuninn --help`.

## Verify the Archive

Use the matching `SHA256SUMS.txt` file shipped with each release.
